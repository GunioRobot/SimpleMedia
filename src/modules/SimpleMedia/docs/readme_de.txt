INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie SimpleMedia in das Verzeichnis modules.
2) Initialisieren und aktivieren Sie SimpleMedia in der Modulverwaltung.
3) Setzen Sie das Verzeichnis `userdata/SimpleMedia/` inklusive aller Unterordner auf beschreibbar.

Bei Fragen und anderen Anmerkungen besuchen Sie unsere Homepage http://zikula.de.
